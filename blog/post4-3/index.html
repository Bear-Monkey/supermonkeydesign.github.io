第四章 展示你的個人網站

By Super Monkey 01 November 2021

4-3.添加 Google Analytics

1.安裝 Google Analytics
  1. 網站搜尋 Google Analytics
  2. 登入 Google 帳號
  • https://accounts.google.com/ServiceLogin/signinchooser?elo=1&flowName=GlifWebSignIn&flowEntry=ServiceLogin
  • sm2021design（Super Monkey）
  1. 帳戶名稱：Super Monkey
  2. 資源名稱：github page 帳號
  • supermonkeydesign.github.io

  1. 開啟「建立通用 Analytics 分析資源」的開關

  2. 填上個人網站連結

  • supermonkeydesign.github.io

  1. 選擇「建立通用 Analytic 分析資源」

  2. 設定：產業別、文章類型、商家規模、GA運用商家，下面選項全部勾選並接受

2.Google Analytics 設定
  1. 勾選「成效建議與最新消息」，每月可收到報表
  2. 資源設定：複製追蹤 ID
3.VS Code 設定 GA
  1. 開啟 config.toml > 將追蹤 ID 填入 googleAnalitycsID
  2. 上傳至 Github Desktop 更新個人網站

主題不支援GA，修改方式

  1. Hugo 搜尋Google Analytics

  2. Use the Google Analytics Template 複製程式碼

{{ template “_internal/google_analytics.html” . }}

  1. VS Code 開啟 >「Themes」>「layouts」>「partials」> head.html

  2. head.html > 將上面程式碼貼入 ＜head＞ tag 裡面

  3. Hugo 複製程式碼 Configure Google Analytics

  • 選擇toml> googleAnalytics = ‘UA-PROPERTY_ID’
  1. 開啟 config.toml > 將上列程式碼貼在下面即可

Google Analitycs

  • googleAnalitycsID = “UA-211224599-1”
  • googleAnalytics = ‘UA-PROPERTY_ID’ 主題本身有支援則無需設定上列googleAnalytics
4.驗證 Google Analitycs 設置成功
  1. 開啟個人網站supermonkeydesign
  2. 開啟GA追蹤程式碼
  3. 成功顯示：狀態> 過去 48 小時內曾收到流量資料。1 位活躍使用者 (目前)。詳情請參閱即時流量報表

GA 報表、指標介紹

1.Google Analytics (分析) 首頁
  • 使用者：期間內觀看者人數
  • 跳出率：期間內瀏覽後離開網站
  • 工作階段持續時間：停留該網站平均多久時間
  • 左下預設7天，可更動
  • 右邊為當下瀏覽人數
2.您如何開發使用者？流量來源
  1. Social：社交網站
  2. Referral：由其他網站，連結進到你的個人網站
  3. Direct：直接於網址列中輸入網址進入
  4. Organic Search：Google 搜尋點擊進入
3.您的使用者位於哪些地區？

瀏覽網站者區域

4.您的使用者會在哪些時段造訪？

瀏覽網站時段，以台灣時間為主

5.您的使用者常使用哪些裝置？

電腦：以該裝置瀏覽網站

6.您的使用者常造訪哪些網頁？

網頁瀏覽次數，次數多寡較熱門

7.長期來看，您的活躍使用者有什麼變化？

以1、7、30天數來計算，期間瀏覽人數

8.您能否留住使用者？

曾造訪該網站回流率

補充：設定 GA 後，無論是本機或上傳後的網站流量，均會被計算列入
