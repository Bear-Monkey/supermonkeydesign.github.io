第二章 認識 Hugo 單元四 VS Code 安裝 個人網站參數設定

By Sindy's Blog 19 October 2021

2-4.VS Code 安裝 個人網站參數設定

1.安裝 VS Code :

  1. 網路搜尋 VS Code

  2. 選取相對應版本安裝 MAC or Windows

  3. VS Code 開啟 > 桌面檔案夾「exampleSite」

2.設定 Hugo 主題

VS code 預設不支援 Hugo，必需透過差件來支援

  1. VS Code 開啟 config.toml

Hugo 主要設定檔 : config.toml

3.程式語言顏色設定

  1. 開啟 VS code 插件管理 左下icon

  2. 搜尋、安裝「toml」來支援 Hugo 程式語言言色 可選擇下載人數多的安裝

回到「config.toml」看到程式碼已有顏色即完成

4. VS Code 程式碼介紹

default configuration : 預設設定

  1. baseURL：修改為個人 github 網址 (https://supermonkeydesign.github.io)

  2. language：zh-tw 台灣語系

  3. title：個人網站名稱 超級猴子設計實驗室

5. 在電腦上運行個人網站

  1. 桌面檔案夾「exampleSite」右鍵選擇 「新增位於檔案夾位置的終端機視窗」

  2. 每異動務必至終端機輸入：hugo server

  3. 終端機複製產出連結，貼至網站即可

    Web Server is available at http://localhost:1313/ (bind address 127.0.0.1) Press Ctrl+C to stop

終止 hugo 運行，到終端機：輸入 Ctrl+C 或直接關掉
