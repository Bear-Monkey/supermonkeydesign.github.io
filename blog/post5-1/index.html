第五章 進階篇：客製化現有主題

By Sindy's Blog 02 November 2021

5-1. 自定義網站版型

HTML 和 CSS 學習資源

  • 動畫互動網頁程式入門 (HTML/CSS/JS) - 吳哲宇
  • W3Schools HTML Tutorial

下列為讓程式碼顯示，「 <_xxx> 」前空一格，正確寫法不可有空格

1. HTML 常見標籤
  • < div>、< a>、< img>
  • < div id=“card” >< /div> 與 < div class=“card” >< /div> 差異
  • Bootstrap Grid 功能
2. CSS 常見標籤
  • .card {} 和 #card {} 差異
  • .card .card-title {} 代表意思

自訂版型修改

Themes 資料夾介紹

  • Themes 資料夾放置：所有 Html & css 相關檔案
  • layouts 資料夾放置：Html
  • asset 資料夾放置：javascript & scss

scss & css 差異：強化版、語法一樣、增加功能

1.首頁版型程式碼修改;刪除近期文章區塊
  1. 開啟 layouts > index.html
  • 第 3 行 feature post 註解下列程式碼顯示該區段
  • 第 40 行 recent post 註解下列程式碼顯示該區段
  • 第 69 行 blog post 註解下列程式碼顯示該區段
  1. 刪除近期文章區塊：第 3 行 feature post 到 第 67 行 feature post 程式碼刪除
2.側邊欄(About Me)由右換置到左邊
  1. blog post 區塊，第 7-9 行 使用 Bootstrap 框架裡面的 Grid 功能

    第 7 行 < div class=“container”>

第 8 行 < div class=“row”>

第 9 行 < div class=“col-lg-8 mb-5 mb-lg-0”>

Hugo 專有語法：雙大括號 {{ }}

  1. 第 39 行 sidebar 註解
  2. 第 40 行 為 sidebar 程式碼

    {{ partial “sidebar.html” . }}

Hugo 會載入放在 layouts 資料夾下的 partials 資料夾內的 sidebar.html 檔案 的程式碼

  1. sidebar 段剪下，貼到 row 下方，側邊欄即右到左換置
  • 因運用 Bootstrap 框架的 Grid 功能，可這樣執行
  • 側邊欄換置位置後，到下面 分類頁面確認內頁是否也同步，如無，需到各別頁面的 html 檔內修改程式碼
3.側邊欄下分類版型修改
  1. themes > layouts >_default > list.html
  2. 將 sidebar 區塊程式碼剪下貼到 row 下即可
4.刪除文章上的圖片區塊 card

  1. 開啟 layouts > index.html

    < article class=“card”>

  2. 刪除第 18-20 行程式碼，即完成刪除文章上的圖檔

  • 第 18 行 hugo 語法，判定 Markdown 檔案中有無寫上 image 設定

    {{ if .Params.Image }}

  • 第 19 行：如 Markdown 有 image 設定，則呈現圖片

    < img src="{{ .Params.Image | absURL }}" class=“card-img-top” alt="{{ .Title | markdownify }}">

  • 第 20 行 結尾

    {{ end }}

5. 日期顯示修改，預設為：英文月份 / 數字年

  1. 第 29 行 .PublishDate.Format，後面為呈現模式

    {{ .PublishDate.Format “02 January 2006” }}

  2. 開啟 Hugo 網站搜尋 hugo date

  3. Hugo Date and Time Templating Reference 下為 hugo 所有支援格式

  4. 於 hugo 上 copy 要換置的日期格式，換置即可

    {{ .PublishDate.Format "2006-01-02" }}

6.文章簡介自定義修改

每篇文章簡介顯示為文章前段內容，修改成 第 10 行 description 簡介

  1. 開啟 themes > layouts >_default > index.html
  2. 第 32 行 .Summary 改成 .Description

    < p class=“card-text”>{{ .Summary }}

< p class=“card-text”>{{ .Description }}

首頁 index.html 改動後，內頁 list.html 也需更動 3. 開啟 themes > layouts >_default > list.html

  1. 第 14 行，載入

    {{ .Render “post” }}

  2. 開啟文章設定檔 themes > layouts >_default > post.html

  3. 第 16 行 日期格式更動與首頁一致

    {{ .PublishDate.Format “02 January 2006” }}

  4. 第 19 行 文章簡介顯示，更動與首頁一致

7. 刪除側邊欄下方 AD Banner 廣告預留位置
  1. 開啟 layouts > partials > sidebar.html
  2. 第 53 行註解，如有申請 Google AdSense 可於此設置

    < !– advertisement –>

  3. 將 advertisement 區塊移除 即不呈現廣告
8. 刪除網頁下方資訊欄 footer

如程式碼中找不到區塊，回到網頁右鍵查詢

  1. 開啟 themes > layouts >_default > index.html

  2. VS Cond 左邊搜尋 <footer>

  3. 開啟 themes > layouts >_default > baseof.html

    {{- partial “footer.html” . -}}

  4. 開啟 themes > layouts > partials > footer.html

  • 第 23-83 行 footer 區塊刪除
