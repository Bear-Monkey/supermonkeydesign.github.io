4-4.添加 Google Search Console

Sitemap 介紹

Sitemap 指個人網站地圖，上傳至 Google Search Console。 Google 知道個人網站所有頁面後，存至 Google 搜尋引擎裡面，在 Google 上搜尋時 就能找到你的個人網站。

1.Google Search Console 安裝

  1. 搜尋 google search console

  2. 立即開始 > 網址前置字元 > 輸入 個人網站連結超級猴子設計實驗室

  3. 驗證 GA：前往資源 > 開始

2.Google Search Console 設定
  1. 點擊左邊： Sitemap
  2. 開啟 VS Code >「pubilc」> sitemap.xml
  3. 開啟終端機
  4. 上傳 Github Desktop
  5. 新增 Sitemap
  • 新增 Sitemap
  • 輸入 https://supermonkeydesign.github.io/`Sitemap.xml` 提交
