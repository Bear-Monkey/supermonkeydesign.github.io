第二章 認識 Hugo 單元二 在MAC上使用Hugo運行個人網站

By Super Monkey 19 October 2021

2-2. 在 MAC 上使用 Hugo 運行個人網站

安裝教學：MAC 單元2-2 / PC 單元2-3

先安裝 Homebrew 套件支援 Hugo，才能再安裝 Hugo

1.Homebrew Mac 版安裝
  1. 網路搜尋「mac brew」

macOS 缺少套件的管理工具 - Homebrew (https://brew.sh/index_zh-tw)

  1. 執行 Homebrew : 複製頁面程式碼貼至終端機

/bin/bash -c “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

安裝成功顯示 : Installation successfu!

2.安裝 Hugo

  1. mac 安裝教學 : Hugo 網站下方複製 mac 程式碼貼到終端機執行brew install hugo

  2. 貼到終端機執行， 即安裝 hugo

3.Homebrew 安裝後輸入：hugo version

安裝成功顯示：Hugo Static Site Generator

4. 下載 Hugo 主題

[Hugo Themes] (https://themes.gohugo.io/)

  1. 下載主題 Livs Hugo

  2. 解壓 livs-hugo-master，內「exampleSite」檔案夾拉到桌面

  3. exampleSite 內新增「Themes」檔案夾

新增檔案夾務必要叫「Themes」

  1. 回到下載內將「livs-hugo-master」移至，桌面「exampleSite」內「Themes」檔案夾

  2. 「Themes」內「livs-hugo-master」刪掉「master」

5. 在電腦上運行個人網站

  1. 桌面檔案夾「exampleSite」右鍵選擇 「新增位於檔案夾位置的終端機視窗」

  2. 每異動務必至終端機輸入：hugo server

  3. 終端機複製產出連結，貼至網站即可

Web Server is available at http://localhost:1313/ (bind address 127.0.0.1) Press Ctrl+C to stop

終止hugo運行，到終端機：輸入 Ctrl+C 或直接關掉
