第四章 展示你的個人網站

By Super Monkey 28 October 2021

4-2.添加評論功能

安裝 Hugo comments 支援靜態的 Github Page

1.Hugo 網站搜尋「comments」

(https://gohugo.io/content-management/comments/#readout)

2.檢測該主題是否支援評論功能
  1. VS Code > 到 config.toml
  2. disqusShortname 即有支援 Disqus 評論
  3. Disqus 帳號填入 disqusShortname 後即可

如無支援自己加入

3.Hugo 網站搜尋「Disqus」

(https://gohugo.io/templates/internal#disqus)

  1. 複製 Use the Disqus Template 下的程式碼

{{ template “_internal/disqus.html” . }}

  1. 檔案夾「Themes」>「layouts」>「default」> single.html

「layouts」html 程式碼檔案 / 「default」html 文章程式碼

  1. 到 下貼上程式碼

部分主題如找不到此檔案，需到「layouts > default > single.html 」自行搜索

Disqus 帳號申請

1.Disqus 安裝設定
  1. 到 Hugo 網站搜尋 Disqus
  2. Disqus > 點擊左上「Get Started」
  3. 可用 google 申請，同 Github 帳號
  1. VS Code 開啟 config.toml
  2. 將 Disqus 的帳號填入

disqusShortname = “super-monkey”

2.Disqus 驗證申請成功
  1. 開啟終端機 > hugo server
  2. 文章下方出現字樣，即表示設定完成

Disqus comments not available by default when the website is previewed locally. comments powered by Disqus

3.Disqus 上傳至 Github page
  1. 關閉終端機 > control + D
  2. exampleSite 標籤頁 > 開啟終端機 > hugo
  3. exampleSite > public 內檔移動到 supermonkeydesign.github.io
  4. 文件 > Github > supermonkeydesign.github.io
  5. 開啟 Github Desktop > commitDisqus> 上傳
  6. 到個人網站文章下確認評論功能即可

尚有其他評論支援教學觀看 Hugo
