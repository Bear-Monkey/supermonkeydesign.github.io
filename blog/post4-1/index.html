第四章 展示你的個人網站 單元一 發佈個人網站至 Github Pages

By Super Monkey 27 October 2021

4-1. 發佈個人網站至 Github Pages

1.Repository 創建：

Github Web 點選右上「+」號 > 選擇「New repository」 Github

2.New repository 命名規則：

github帳號名稱後，輸入github.io，設定完成到下面點選「creative repository」送出後，產生檔案夾資訊頁面

  • supermonkeydesign.github.io

  • go-travel.github.io

3.Repository clone 到電腦端：

點選左上「Set up in Desktop」開啟「GitHub Desktop.app」

4.Clone a Repository 位置：

預設位置「Finder」建議不要變動 點選「Clone」

/Users/sindysu/Documents/GitHub/sindy528.github.io

5.Github Desktop 新增：

新增檔案夾：Clone 成功可看到「Current Repository」下顯示新資料夾名稱

6. 檔案夾本機位置：
  1. Finder > 文件 > Giuhub > 產出新檔案夾「supermonkeydesign.github.io」

Hugo 產生個人網站程式碼

電腦上運行Hugo個人網站，終端機輸入hugo server

Hugo 產生個人網站程式碼，終端機輸入hugo

1.Hugo 產出個人網站程式碼：

終端機輸入hugo看到圖表即表示成功

異動至終端機輸入：hugo，出現 Total in 210 ms 顯示產出時間方可繼續動作

2.Hugo 產出個人網站程式碼：
  1. 桌面「exampleSite」> 產出新資料夾「public」
  2. 將「public」內拉 到 Finder > 文件 > Github > 「supermonkeydesign.github」檔案夾內
3.上傳程式碼到 Github：
  1. 開啟 Github Desktop
  2. Changes 下確認檔案夾，輸入：Commit
  3. 點選「Commit to main」
  4. 右上「Fetch origin」顯示異動檔案數，確認上傳更新
4. 確認更新

  • Giehub 個人網站

  • Hugo 個人網站

新增Blog文章

1.VS Code
  1. content > blog 右鍵新增
  2. 檔名後，副檔名.md
  3. 開舊檔複製 Hugo文章設定，貼到新章上

Hugo 文章設定：見文章1~24行程式碼

2. Hugo 產出程式碼上傳 Github
  1. 終端機輸入：hugo
  2. 桌面 exampleSite > public 檔案拖拉到「文件」中「github.io 」資料夾
  3. 文件 > github > github.io
  4. Github Desktop > commit > Commit to master > Fetch origin
  5. github 重新整理頁面，個人網站更新完成
