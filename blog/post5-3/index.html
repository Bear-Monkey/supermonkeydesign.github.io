By Super Monkey 02 November 2021

5-3. 自定義 Markdown 渲染效果 reminder

自定義文章內容的 css 樣式的時候，css 樣式的影響範圍縮小。例如：liva-hugo 主題，所有的 Markdown 文章內容，放至 content 的 div 裡面。在自定義 Markdown 的渲染效果的時候 最好在前面加上一個 .content，這樣 css 改動 才不會影響到個人網站的其他地方！

1.文章呈現寬度修改

  • 開啟 themes > layouts > _default > single.html
  • Bootstrap 框架的 Grid 功能

< div class=“container”> < div class=“row”> < div class=“col-lg-8 mx-auto”> 改成 < div class=“col-lg-10 mx-auto”>

2.標題呈現修改

  • 開啟 themes > assets > scss > templates > _main.scss
  • 第 231 行起 /* markdown style */，為設定 Markdown 文章的渲染效果
  • .content 的 class 裡面加上一行 font-family 的設定

.content {

  • { font-family:sans-serif; margin-bottom: 20px; } 如要另外載入建議使用 google fonts

3.google fonts 使用方法

  • 搜尋 google fonts
  • 進入後範例：搜尋 flower
  • 點擊字型，進入後再點擊右邊 Selected family
